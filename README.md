# blog
基于Django的一个博客系统
## 主要功能  
* 首页显示所有文章的标题，简介，分类，标签，时间以及分页。
* 文章及页面支持Markdown。
* 支持文章标题搜索。
* 支持文章微信，微博，QQ空间等分享。
* 侧边栏功能：分类，归档，近期文章。
